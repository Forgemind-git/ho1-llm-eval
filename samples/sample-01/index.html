Support Email Benchmark

Evaluate how well each model drafts a professional, empathetic customer support reply.

Problem Statement

You are a customer support agent. A customer wrote: "I ordered the Premium Plan 5 days ago but my dashboard still shows Free." Draft a professional, empathetic reply.

What You're Measuring

CriterionWhat to look forScale
Tone & empathyDoes the model acknowledge the frustration and show genuine care?1–5
Accuracy — addresses the real issueDoes the reply focus on the billing/upgrade problem, not generic fluff?1–5
Length & scannabilityIs the reply concise with clear paragraphs, not a wall of text?1–5
Call-to-action clarityIs there a clear next step for the customer to take or await?1–5

Run the Test

No API key needed. Run this test manually in two tools and compare results.
  1. LM Studio Open LM Studio on your laptop. Load a model (e.g. Mistral-7B or Llama-3.1-8B). Go to the Chat tab.
  2. LM Studio Paste the prompt above (starting with "You are a customer support agent…") into the chat and press Send. Copy the full response into the Comparison Table below.
  3. Claude.ai Open claude.ai in a new tab. Start a new conversation.
  4. Claude.ai Paste the same prompt and press Send. Copy the full response into the Comparison Table below.
  5. Score Use the scorecard below to rate each model on the four criteria.
